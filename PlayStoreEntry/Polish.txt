Root ToolCase

SHORT DESCRIPTION
*****************

Kolekcja świetnych narzędzi, która pozwoli wyciągnąć jak najwięcej z Twojego urządzenia!

FULL DESCRIPTION
****************

<b>ROOT WYMAGANY (aplikacja nie instaluje root'a)</b>

<b>ZASTRZEŻENIE:</b>
Aplikacja posiada potężne możliwości, bądź ostrożny podczas korzystania z nich.
TWÓRCA APLIKACJI NIE JEST ODPOWIEDZIALNY ZA EWENTUALNE PROBLEMY Z URZĄDZENIEM!

<b>App Manager:</b>
★ czyszczenie danych i cache'u aplikacji
★ włączanie/wyłączanie (zamrażanie) aplikacji
★ usuwanie aplikacji

<b>Flash Kreator:</b>
★ instalacja z plików zip
★ tworzenie kolejki instalacji dla kilku plików zip
★ instalacja obrazów boot (kernel)
★ instalacja obrazów recovery

<b>Init.d Emulator:</b>
★ uruchamianie skrytpów podczas startu
★ natywne wsparcie dla init.d nie jest wymagane

<b>Build.prop Edytor:</b>
★ edytuj build properties swojego ROM'u

<b>Zaawansowany Reboot:</b>
★ uruchom recovery lub bootloader (fastboot)

<b>Widgets:</b>
★ czyszczenie zadań
★ Zaawansowany Reboot
